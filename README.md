# Book-Store-Website-Project
The Book Store Website Project is a full-stack web application designed to provide an online platform for browsing, searching, and purchasing books. It features user authentication, category-wise book listings, shopping cart functionality, order management, and an admin panel to manage inventory, users, and orders efficiently.
